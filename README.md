# Yelp Data Exploration
This is a basic Data exploration of Yelp Dataset 

# Problem Statement
To find insights from raw data which contains information regarding restaurants and their reviews.

# Data source
The link for Dataset is : https://www.yelp.com/dataset

# Approach
This Data Analysis is done using raw data and finding specific tables from MySQL Workbench.
The tables are pulled out for the specific questions subject to constraints.
An attempt to find similarities to generate meaningful insights

# Application
This type of analysis can be used for understanding data of restaurants to further improve decision making
This analysis can also be used to understand the customer sentiments and take appropriate decisions to improve customer satisfaction.

Feel free to use the SQL codes and test them yourself.
